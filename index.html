The Modern Semantic HTML Tags and Attributes Table

Semantic HTML introduces meaning to a page through specific elements that provide context as to what is in between the tags.

Semantic HTML Tags

Tag Name Description
<header> Header creates the basic structure of the webpage, is a container usually for either navigational links or introductory content containing <h1> to <h6> headings.
<nav> Navigation creates the basic structure of the webpage, is used to define a block of navigation links such as menus and tables of contents. It is important to note that tag can be used inside of the <header> element but can also be used on its own.
<main> Main creates the basic structure of the webpage. By using this tag as opposed to a <div> element, screen readers and web browsers are better able to identify that whatever is inside of the tag is the bulk of the content.
<footer> Footer creates the basic structure of the webpage. The footer contains information such as: Contact information, Copyright information, Terms of use, Site Map, Reference to top of page links
<section> Section defines elements in a document, such as chapters, headings, or any other area of the document with the same theme.
<article> Article holds content that makes sense on its own such as articles, blogs, comments, etc.
<aside> Aside contains information that is related to the main content, but not required in order to understand the dominant information.Some common uses of the element are for: Bibliographies, Endnotes, Comments, Pull quotes, Editorial sidebars, Additional information
<figure> Figure is an element used to encapsulate media such as an image, illustration, diagram, code snippet, etc, which is referenced in the main flow of the document.
<figcaption> Figcaption is used to describe the media in is an element used to describe the media in the <figure> tag. Usually, this tag will go inside the <figure>.
<audio> Audio elements are used for media files.The tag element is used to embed audio content into a document. Like <video>, <audio> uses src to link the audio source.
<video> Video elements are used for media files.
<embed> Embed the tag, which can embed any media content including videos, audio files, and gifs from an external source. This means that websites that have an embed button have some form of media content that can be added to other websites. The tag is a self-closing tag, unlike the <video> element.
<source> Source this element encapsulates audio link together with src="", tag is using with <audio>

Attributes

Attribute Name Description
controls Controls attribute When added in <video>, a play/pause button will be added onto the video along with volume control and a fullscreen option. When added in <audio>, automatically displays the audio controls into the browser such as play and mute.
autoplay Autoplay attribute The attribute which results in a video automatically playing as soon as the page is loaded.
loop Loop attribute This attribute results in the video continuously playing on repeat.
